# Workplace Decision Engine

This project simulates a workplace decision-making environment using OpenEnv.

## Features
- Task prioritization
- Action selection (complete, respond, schedule)
- Reward-based evaluation

## Endpoints
- `/reset` → Reset environment
- `/step` → Perform action
- `/run-demo` → Run full simulation
